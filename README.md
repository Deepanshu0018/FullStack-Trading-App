# FullStack-Trading-App

A full-stack stock trading web application inspired by Zerodha, built with **React**, **Node.js**, and **Express**. Includes a **dashboard**, **frontend**, and **backend** for portfolio management, watchlists, and analytics.

---

## 📁 Project Structure


---

## 🚀 Features

- User Authentication (Signup/Login)
- Dashboard with Holdings, Orders, Positions
- Watchlist with Buy/Sell/Analytics quick actions
- Portfolio tracking with P&L calculation
- Interactive charts using Chart.js
- Responsive UI with Bootstrap & Material UI

---

## 💻 Technologies Used

- **Frontend:** React, React Router, Chart.js, Material UI, Bootstrap  
- **Backend:** Node.js, Express, Axios  
- **Database:** (Add your DB, e.g., MongoDB, MySQL)  
- **Styling:** Bootstrap, custom CSS
