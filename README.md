# Full-Stack Trading App

A full-stack trading platform clone with backend, frontend, and dashboard modules. Features portfolio tracking and order simulation for learning and practice in web development.

## 🚀 Features
- Portfolio tracking and analytics
- Simulated buy & sell orders
- Interactive dashboard for stocks
- Responsive frontend interface

## 🛠️ Tech Stack
- **Frontend:** React.js / JavaScript / HTML / CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Dashboard:** Custom analytics module
